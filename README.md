# e-commerce-store-testing

Based on course from Udemy by Neil Cummings.

To use, navigate to ecommerce-testing folder:

Update appsettings.Development.json

To start the API : 
```
cd API
dotnet run
```

To start client-side:
```
cd client
npm install
npm start
```
